HTML

HTML significa hyper text markup language y es utilizado para darle estructura a las paginas web. HTML se guarda .html y la primera linea que se debe poner al usar este codigo es: !DOCTYPE html seguido de html y /html, dentro de esto se debera escribir el codigo

Elementos y atributos

En html existen elementos y atributos. Los elementos estan envueltos en signos de mayor o menor y generalmente tienen un elemento que abre y otro que cierra (tagname-contenido-/tagname). Los elementos definen la accion que debe hacer el navegador. Un elemento de html puede estar dentro de otro, por ejemplo dentro del html va el body y dentro el parrafo.

No todos los elemetos llevan endtag pero no ponerlo puede ocasionar errores, ademas es recomendable escribir siempre en minusculas.

Todos los elementos de HTML pueden tener atributos, estos proveen informacion extra sobre el elemento que acompa√±an. Los atributos deben ser escritos junto con la start tag y respecto a su sintaxis generalmente es name="value".

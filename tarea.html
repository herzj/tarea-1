Sobre lo aprendido

Lo primero que hay que hacer es crear una cuenta en github y
instalar git en tu computadora, git sera el respositorio en el que se guardaran los proyectos.
A continuacion se creara un projecto en la pagina de git con licencia abierta.
Despues se debera copiar el link del proyecto.
Lo siguiente es abrir la terminal del ordenador y posicionarte donde se quiere crear la carpeta para el proyecto, por ejemplo
si es el escritorio se ingresa a la terminal "cd Desktop"
A continuacion se escribe git clone y el link del proyecto git, esto creara una carpeta con los datos del git
En la carpeta se guardan los archivos del proyecto como este html
Una vez hecho esto se deben subir los cambios a github mediante el uso de la terminal y el comando git add . con el punto
En cualquier momento se puede checar el status de la carpeta con el comando git status
Es recomendable despues de modificar el git anadir una bandera con la explicacion de la modificacion, esto se hace con el comando git commit -m "tu comentario"
El prceso se finaliza con el comando git push

The target attribute can have one of the following values:

		_blank - Opens the linked document in a new window or tab

		_self - Opens the linked document in the same window/tab as it was clicked (this is default)

		_parent - Opens the linked document in the parent frame

		framename - Opens the linked document in a named frame
